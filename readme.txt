Add a side menu to your theme to display and control cart items.

== Description ==

This plugin adds a side menu to your theme to display and control cart items for more elegant cart management.

== Installation ==

1. Upload the plugin to the **/wp-content/plugins/** directory
2. Activate the plugin through the 'Plugins' menu in WordPress
